# Nasıl Kullanacağım?

Bilgisayarınıza Klonlayın

```bash
  git clone https://github.com/knqsec/dorkcreator.git
```

Dosyaya Gidin

```bash
  cd dorkcreator-main
```

Toolu Çalıştırın


```bash
  py knqdorkcr.py
```
